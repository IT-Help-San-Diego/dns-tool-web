{{icon "question-circle" "me-2"}}Where to Find Your Scan Number

After running a scan, look at the URL in your browser address bar:

dnstool.it-help.tech/analyze?domain=example.com&id=1234

The number after id= is your scan number. Using a scan number pins the badge to that specific scan result.

Using a domain name always shows the most recent public scan.

{{icon "info-circle" "me-2"}}Badge Styles
Detailed — Full security assessment card showing posture score (0–100), protocol-by-protocol breakdown (SPF, DKIM, DMARC, DNSSEC, DANE, MTA-STS, TLS-RPT, BIMI, CAA), risk classification, and scan date. Best for documentation, security pages, and README files where you want to show comprehensive posture.
Compact — Standard shields-style badge showing risk level with color coding. Quick visual indicator for GitHub profiles, blog sidebars, and inline references.
Covert — Terminal-aesthetic badge with monospace font and dark red palette. For security researchers and red-team documentation.
